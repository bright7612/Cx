  • 献礼十九大，工会为基层职工送去一份文化大餐！

    2018-03-26

    10月20日晚，湖州市总工会职工艺术团下基层慰问演出在和平浙江天能动力能源有限公司蓝领公寓成功举办。此次活动由文化中心湖州市总工会主办，长兴县总工会、湖州市工人文化宫协办，通过歌舞、曲艺、器乐等多种表演形式，对我县十九大召开、社会经济建设发展的成果进行了宣传。

    县总工会党组成员黄金华、鲍强华，和平镇总工会副主席黄美，天能集团党委副书记章旭东，党群部主任汤林强，浙江天能动力能源有限公司党支部委员、工会副主席尹明成等领导出席并观看慰问演出。
